# Nix templates

A few Nix project templates to get started using Nix in your projects or start a new project.

## How to use
Run the following in a shell:
```bash
mkdir new-project
cd new-project
nix flake init -t github:mith/nix-templates#bevy
```

See the [Nix cli documentation](https://nixos.org/manual/nix/stable/command-ref/new-cli/nix3-flake-init.html) for more details.
